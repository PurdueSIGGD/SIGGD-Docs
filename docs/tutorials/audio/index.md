# Audio Tutorials

Topics covered here include:

- Recording, synthesis, editing in Audacity, quick SFX with Bfxr/ChipTone
- Writing game music in a DAW, looping cleanly, genre-appropriate styles
- LMMS, GarageBand, or other DAWs: tracks, MIDI, mixing, exporting
- AudioSource/AudioClip, AudioMixer, spatial 3D audio, 2D UI sounds (Unity)
- AudioStreamPlayer, AudioBus, positional audio (Godot)
- FMOD setup with Unity: events, parameters, adaptive audio
- Music that reacts to gameplay state, layered stems, dynamic mixing
- Reverb, effects, ducking, footsteps, impact weight
- Audio compression, streaming vs. preloaded, memory considerations
