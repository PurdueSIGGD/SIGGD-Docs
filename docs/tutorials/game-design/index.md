# Game Design Tutorials

Topics covered here include:

- Primary loops, secondary loops, feedback cycles, player motivation
- Flow, pacing, spatial storytelling, difficulty curves, tutorialization
- What to include in a GDD, keeping it collaborative and lean
- Prototyping mechanics, verbs and nouns, emergence and depth
- Running a playtest session, what to observe vs. what to ask
- Skill curves, rubber-banding, accessibility options
- Environmental storytelling, dialogue trees, ludonarrative coherence
- Affordances, feedback, information hierarchy in game UI
- Paper prototypes, grey-box levels, rapid iteration
- Scoping for 48 hours, cutting features, shipping something playable
