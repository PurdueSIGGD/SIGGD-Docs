# About SIGGD

**SIGGD** (Special Interest Group in Game Development) is a student
organization at Purdue University dedicated to learning and practicing game
development in a collaborative, inclusive environment.

We welcome students of **all skill levels and disciplines** — programmers,
artists, designers, audio engineers, writers, and anyone who's ever wanted
to make a game.

---

## What We Do

- **Weekly meetings** — workshops, jam kickoffs, and project work sessions
- **Game jams** — we participate in Ludum Dare, Global Game Jam, and run our own internal jams
- **Semester projects** — longer-form games built collaboratively across a semester
- **Workshops** — skill-building sessions on Unity, art tools, design principles, and more
- **This documentation site** — so nothing we learn gets lost
