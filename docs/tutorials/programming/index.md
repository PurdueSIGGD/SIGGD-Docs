# Programming Tutorials

Topics covered here include:

- Unity setup, Godot setup, first scripts, project structure
- C# and GDScript fundamentals, MonoBehaviour lifecycle, Nodes
- Movement, input handling, cameras, physics, animation state machines
- Health, inventory, save/load, scene management, game managers
- Patrol, detection, chase, pathfinding with NavMesh / NavigationAgent
- HUDs, menus, pause screens, ScriptableObject data binding
- Shaders in HLSL / ShaderGraph / Godot's visual shader editor
- Singleton, Observer, State Machine, Object Pooling, ECS concepts
- Procedural generation: tilemaps, dungeons, noise-based terrain
- Profiling, batching, draw calls, garbage collection
