# External References

---

## Programming

### Unity Documentation

**Link:** [docs.unity3d.com](https://docs.unity3d.com)

The official Unity manual and scripting API. Always check the documentation
version matches your Unity version.

---

### Game Programming Patterns

**Link:** [gameprogrammingpatterns.com](https://gameprogrammingpatterns.com)

Free online book by Robert Nystrom. The definitive reference for common
design patterns in game code (state machines, observers, object pools, etc.).
Essential reading for any game programmer.

---

### Refactoring Guru

**Link:** [refactoring.guru](https://refactoring.guru)

Online site explaining design patterns in depth with code and diagram examples.
Important resource for a lot of the systems made in this club.
