# Game Design Tutorials

The craft of making games fun — from high-level theory to practical tools
you can apply in a game jam this weekend. Game design is for everyone on the
team, not just dedicated designers.

---

## Topics Covered Here

This section is home to tutorials on subjects like:

- **Core game loops** — primary loops, secondary loops, feedback cycles, player motivation
- **Level design** — flow, pacing, spatial storytelling, difficulty curves, tutorialization
- **Writing a GDD** — what to include, what to skip, keeping it collaborative and lean
- **Mechanics design** — prototyping an idea, verbs and nouns, emergence and depth
- **Playtesting** — how to run a playtest session, what to observe vs. what to ask
- **Difficulty & balancing** — skill curves, rubber-banding, accessibility options
- **Narrative design** — environmental storytelling, dialogue trees, ludonarrative coherence
- **UI/UX principles** — affordances, feedback, information hierarchy in game UI
- **Prototyping** — paper prototypes, grey-box levels, "is this fun in 20 minutes?"
- **Game jam strategy** — scoping for 48 hours, cutting features, shipping something playable
- **Postmortems** — how to run a postmortem, what went well, what to improve
