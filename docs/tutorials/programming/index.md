# Programming Tutorials

Everything you need to write game code — from setting up your first Unity project
to architecting complex game systems. Tutorials here cover both **Unity (C#)** and
**Godot (GDScript/C#)**, with engine-agnostic programming concepts mixed in.

---

## Topics Covered Here

This section is home to tutorials on subjects like:

- **Getting started** — Unity setup, Godot setup, first scripts, project structure
- **Core scripting** — C# and GDScript fundamentals, MonoBehaviour lifecycle, Nodes
- **Player systems** — movement, input handling, cameras, physics, animation state machines
- **Game systems** — health, inventory, save/load, scene management, game managers
- **Enemy & AI** — patrol, detection, chase, pathfinding with NavMesh / NavigationAgent
- **UI systems** — HUDs, menus, pause screens, ScriptableObject data binding
- **Shaders** — shader basics in HLSL / ShaderGraph / Godot's visual shader editor
- **Patterns** — Singleton, Observer, State Machine, Object Pooling, ECS concepts
- **Procedural generation** — tilemaps, dungeons, noise-based terrain
- **Performance** — profiling, batching, draw calls, garbage collection
