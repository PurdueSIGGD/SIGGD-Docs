# Audio Tutorials

Game audio is one of the most underrated ways to make a game feel polished.
This section covers the full pipeline from **creating audio** to **implementing
it in your engine**, including middleware like FMOD.

---

## Topics Covered Here

This section is home to tutorials on subjects like:

- **Sound effects** — recording, synthesis, editing in Audacity, quick SFX with Bfxr/ChipTone
- **Music composition** — writing game music in a DAW, looping cleanly, genre-appropriate styles
- **DAW basics** — LMMS, GarageBand, or any other DAW: tracks, MIDI, mixing, exporting
- **Audio in Unity** — AudioSource/AudioClip, AudioMixer, spatial 3D audio, 2D UI sounds
- **Audio in Godot** — AudioStreamPlayer, AudioBus, positional audio
- **FMOD with Unity** — setup, events, parameters, adaptive/dynamic audio
- **Adaptive audio** — music that reacts to gameplay state, layered stems, dynamic mixing
- **Audio polish** — reverb and effects, ducking, footsteps, impact weight
- **Performance** — audio compression, streaming vs. preloaded, memory considerations
- **Voice over** — recording dialogue, noise reduction, implementation patterns
