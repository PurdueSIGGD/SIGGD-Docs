# External References

Curated external learning resources organized by discipline. These are the
sites, channels, and docs that SIGGD members actually use.

---

## Programming

### Unity Documentation

**Link:** [docs.unity3d.com](https://docs.unity3d.com)

The official Unity manual and scripting API. Always check the documentation
version matches your Unity version.

---

### Godot Documentation

**Link:** [docs.godotengine.org](https://docs.godotengine.org)

Comprehensive and well-written official docs. The "Your first 2D game" and
"Your first 3D game" tutorials are excellent starting points.

---

### Game Programming Patterns

**Link:** [gameprogrammingpatterns.com](https://gameprogrammingpatterns.com)

Free online book by Robert Nystrom. The definitive reference for common
design patterns in game code (state machines, observers, object pools, etc.).
Essential reading for any game programmer.

---

### Catlike Coding (Unity Tutorials)

**Link:** [catlikecoding.com/unity/tutorials](https://catlikecoding.com/unity/tutorials)

In-depth Unity tutorials covering rendering, procedural generation, and
advanced game systems. Best suited for intermediate-to-advanced programmers.

---

## Art & Design

### GDC Talks (YouTube)

**Link:** [youtube.com/@Gdconf](https://www.youtube.com/@Gdconf)

The official GDC YouTube channel. Thousands of talks from game developers
on every topic. The "Vault" on gdcvault.com has even more.

---

### Brandon James Greer (Pixel Art)

**Link:** [youtube.com/@BJGpixel](https://www.youtube.com/@BJGpixel)

Excellent YouTube tutorials on pixel art techniques, color theory, and
animation. Great for beginners and intermediate artists.

---

### Blender Guru

**Link:** [youtube.com/@blenderguru](https://www.youtube.com/@blenderguru)

The classic "Donut Tutorial" and many others. Best starting point for
learning Blender. Covers modeling, shading, and rendering.

---

## Game Design

### Design Doc (YouTube)

**Link:** [youtube.com/@DesignDoc](https://www.youtube.com/@DesignDoc)

Video essays analyzing game design decisions in popular games. Great for
developing a vocabulary for game design thinking.

---

### Mark Brown / Game Maker's Toolkit

**Link:** [youtube.com/@GMTK](https://www.youtube.com/@GMTK)

Deep-dives into game design concepts with excellent examples. One of the
best channels for game design theory.

---

### The Game Design Round Table (Podcast)

**Link:** [thegamedesignroundtable.com](http://thegamedesignroundtable.com)

Podcast discussions between working game designers. Good for understanding
how design decisions are made in practice.

---

## Audio

### Game Audio Learning

**Link:** [gameaudiolearning.com](https://gameaudiolearning.com)

Free resources for learning game audio implementation, FMOD, and Wwise.
Covers both the technical and creative sides of game audio.

---

### Ask Audio — Game Audio Section

**Link:** [ask.audio/articles/category/game-audio](https://ask.audio/articles/category/game-audio)

Articles and tutorials on game audio, DAWs, and sound design for games.

---

## General Game Development

### Brackeys (Unity, archived)

**Link:** [youtube.com/@Brackeys](https://www.youtube.com/@Brackeys)

The most popular Unity tutorial channel (now archived, but still fully
accessible). Excellent beginner-to-intermediate tutorials on most Unity topics.

---

### GDQuest (Godot)

**Link:** [gdquest.com](https://www.gdquest.com)

High-quality free and paid Godot tutorials. Their free YouTube content and
open-source demos are particularly valuable.
