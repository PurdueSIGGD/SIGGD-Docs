# Projects

Explore every game and project built by SIGGD members. Each project entry
includes a playable overview, full codebase documentation, team credits,
and a screenshot gallery.

The codebase docs are designed as a **programmer reference guide** — new
members can understand a project's architecture and systems without reading
all the source code.
